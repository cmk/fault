fault

The Fault type is Either SomeException a -> b — a composable exception handler with Arrow, Category, and profunctor instances. Combine handlers with >>>, override for specific exceptions with \<!\>, and run with runFault or withFault.

Also provides: type-safe throw/throwIO distinction (NotInIO), call stack attachment (CallStackException), displayException fix (DisplayedException, displayExceptions), and structured error lifting (Defect).
