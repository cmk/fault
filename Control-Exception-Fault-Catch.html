Description

Monadic exception handling and resource cleanup.

Extends Control.Exception.Fault.Type with MonadUnliftIO-lifted catching, masking, bracket, and Fault evaluation in IO.

All catching functions only catch synchronous exceptions by default.

withFaultBracket :: MonadUnliftIO m => Fault a b -> m r -> (r -> m c) -> (r -> m a) -> m b Source #

Run a Fault handler with bracket-style resource management.

The acquire and release run outside the handler; the body runs through the Fault.

withFaultBracket myHandler
  (openFile "out.log" WriteMode)
  hClose
  (\h -> hPutStr h result)
