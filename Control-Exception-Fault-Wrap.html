Description

Exception wrappers for enriching, converting, and lifting exceptions.

These newtypes wrap exceptions to add context (call stacks, annotations), convert between sync/async, lift arbitrary types into the exception hierarchy, or fix GHC's display behavior.

newtype DisplayedException a Source #

A zero-overhead wrapper that makes GHC use displayException instead of show when printing uncaught exceptions.

Unlike AnnotatedException, this does not add any information to the exception — it only fixes GHC's rendering behavior. Use displayExceptions to wrap main.

main = displayExceptions actualMain

See Why doesn't GHC use my displayException method?
